Main Entities

User

The User entity represents individuals who use the social media automation system. Each user has unique identifiers, a username, and various settings.

Bot

The Bot entity represents automated accounts that interact with users on behalf of a real user. Bots have specific functionalities and can schedule posts and respond to interactions.

Post

The Post entity represents content created by users or bots that is published on the social media platform. Each post has content, a timestamp, and associations with the user or bot that created it.

Interaction

The Interaction entity captures user interactions with posts, such as likes, comments, and shares. Each interaction is tied to a specific user and a specific post.

Schedule

The Schedule entity represents planned posts that are set to be published automatically at a later time. This entity keeps track of when each post should be made and any associated settings.

Account

The Account entity represents the social media accounts linked to the users and bots. This includes authentication details and connectivity settings to various platforms.

Relationships:

  • User can have multiple Bots.
  • Bot can create multiple Posts.
  • User can create multiple Posts.
  • Post can have multiple Interactions.
  • User can have multiple Accounts.
  • Bot can have multiple Accounts.
